[필수 원칙 2] 한자어 주석 (한국 독자의 한자 약화 고려)
═══════════════════════════════════════════════
- 모든 한자어·옛 표현·정치/법률/학술 용어에 첫 등장 시 독음과 뜻을 병기
- 인라인 표기 형식: 漢字(독음 — 한 줄 뜻 풀이)
  · 예: 信託統治(신탁통치 — 다른 나라가 대신 다스림)
  · 예: 領袖(영수 — 한 집단의 최고 지도자)
  · 예: 顧問政治(고문정치 — 외국인 顧問을 요직에 앉혀 실질 통치)
- HTML 구현: <span class="h"> + <small>로 점선 밑줄 + 작은 보조글
- 핵심 한자어 5~10개를 모은 "📖 자주 나오는 한자어" 사전 박스를 도입부 또는 관련 섹션에 배치
- 인명·국명도 첫 등장 시 한자 병기: 蔣介石(장개석), 中華民國(중화민국)
- 통계 항목명도 한자 병기: 사망(死亡), 부상(負傷), 구금(拘禁) 등

[필수 원칙 3] 비교·대조의 비대칭성 강조
═══════════════════════════════════════════════
원문에 항목 A,B,C,D 비교가 등장하면 단순 표 나열로 끝내지 말 것.
다음 단계를 모두 거칠 것:

(1) 각 항목이 글자 그대로 의미하는 바 표시
(2) 각 항목의 실제 의미·범위를 한 줄로 풀어쓰기
(3) 항목 간 차이가 본질적으로 무엇을 의미하는지 분석
(4) 한 항목이 특별히 위험·중요·이질적이라면 시각적으로 분리
    - 더 큰 카드, 다른 색, "⚠️ 트로이의 목마" 식 라벨
    - 위치·크기로 비대칭 강조
(5) 차이의 함의를 "A = B = 결국 C" 등식 형태로 명시
(6) 일반인이 이해 가능한 비유·풀이 추가

예시: "재정·수송·보건·치안" 4국 분담안의 경우
→ 재정/수송/보건은 부서 하나의 일, 치안은 內務部 全部를 의미
→ 시각적으로 3:1 분리, 마지막에 "治安 = 內務部 = 實質 統治" 등식

- 범례(legend) + 캡션 박스("💡 이 지도가 말하는 것") 반드시 추가
- 클레임 영역 안/밖 국가를 시각적으로 구별 (밖이면 흰색·연한 색)

[필수 원칙 5] 시각적 입체감 (단순 줄글 X)
═══════════════════════════════════════════════
다음 요소를 본문 흐름에 따라 적극 사용:

▸ 영웅(Hero) 섹션: 100vh, 그라데이션, 거대 한자 배경 워터마크, 취소선 활용한 반전 제목
▸ 섹션 번호 + 영문 태그 + 한글 부제 (PPT 챕터 스타일)
▸ 카드 컴포넌트: 일기 카드, 학자 분석 카드, 신문기사 카드 (각각 다른 디자인)
▸ 인용 박스: 출처별로 다른 색상 (학자=네이비, 원문=크림슨, 일기=골드)
▸ 채팅 말풍선 대화: 카톡·라인·DM·왓츠앱 스타일 (좌우 교대, 순차 등장)
▸ 타임라인: 세로 라인 + 도트 + 연도 + 사건 설명
▸ 통계 배너: 큰 숫자 + 항목명, 그리드 자동 정렬
▸ 풀쿼트(pullquote): 위아래 선, 굵은 명조체, 가운데 정렬
▸ 충격 박스(yikes): 어두운 배경 + 골드 텍스트 + 위아래 선
▸ 통찰 박스(insight): "핵심 통찰" 라벨 + 골드 톤
▸ 증언 블록(testimony): 핏빛 배경 + "증언" 태그
▸ 비교 그리드 + ≡ 등호 기호로 동일성 시각화
▸ 결론 섹션: 어두운 배경 + 골드 텍스트, 박스 인용

- 스크롤 인디케이터 (히어로 하단 bouncing arrow)
- 호버·터치 시 미세한 변화 (그림자, 색상)
- 과한 모션 금지: 멀미 유발 X, 접근성 고려

[필수 원칙 8] 구조 흐름
═══════════════════════════════════════════════
일반적인 섹션 순서 (원문 성격에 따라 가감):
  1. HERO — 충격적 제목, 취소선·강조로 반전 메시지
  2. QUESTION — 도입 질문, 통념 vs 진실
  3. EVIDENCE — 1차 사료(일기·문서·인용)
  4. CONTEXT — 지도·배경 설명
  5. CORE TENSION — 비교·대조의 핵심 (트로이의 목마형 분석)
  6. DRAMATIC MOMENT — 채팅 말풍선 또는 결정적 장면 재구성
  7. CONNECTING DOTS — 흩어진 사건들이 모이는 카드 시리즈
  8. TIMELINE — 시간순 정리
  9. CONCLUSION — 어두운 배경 + 핵심 한 문장
  10. FOOTER — 출처, 원문 링크

각 섹션은 .reveal 클래스 + 섹션 번호(01~) + 영문 태그 + 한국어 제목.
